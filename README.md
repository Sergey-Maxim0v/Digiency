### Цель проекта тренировка:
- точное соответствие дизайну,
- адаптивная верстка,
- SCSS,
- использование $переменных,
- использование @import,
- CSS transition
- CSS flex
- CSS псевдоклассы

Проект собран при помощи Prepros.

Сайт адаптирован под ширину экрана: от 1024px, до 1024px, до 600px.
